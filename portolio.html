App Móvil

Entrena+ Atletismo

2023-2024 | Proyecto Final Universitario

App móvil creada en React Native para el proyecto final de Tecnicatura en Programación. La aplicación permite consultar datos actualizados de la CADA (Confederación Argentina de Atletismo) y gestionar entrenamientos, ejercicios y grupos de entrenamiento para atletas y entrenadores.

Funcionalidades

Rankings CADA Resultados de Torneos Calendario Anual Crear Ejercicios Crear Entrenamientos Grupos de Entrenamiento Calendario Grupal Calendario Individual Agendar Entrenamientos Agendar Competencias Gestión de Administrador Resultados de Entrenos

La app permite consultar información actualizada de la CADA incluyendo rankings, resultados de torneos y calendario anual. Los usuarios pueden crear ejercicios y entrenamientos, formar grupos de entrenamiento, y gestionar calendarios grupales e individuales para agendar entrenamientos y competencias. Los administradores de grupo pueden agendar entrenos y completar resultados de entrenos de los participantes.

Tecnologías

React Native Java Spring Boot REST APIs

Frontend móvil construido con React Native para iOS y Android. Backend desarrollado con Java y Spring Boot para gestionar la comunicación con la API de la CADA y las funcionalidades de entrenamiento y grupos.
